# Project Wonderland Roadmap

This is the retained product backlog. Sprint 1 implements only the framework bootstrap.

## Foundation

- Framework bootstrap, configuration, persistence, logging, module API, events, networking, and validation tools.
- Standalone design: no Expansion dependency.

## Living bases

- Buildable storage: lockers, cabinets, dressers, gun cabinets, refrigerators, freezers, beds, couches, tables, shelving, and workshops.
- Placement, ownership, permissions, persistence, repair, and raid-damage rules.
- Solar/generator/battery power, lights, wiring, fuse boxes, water collection, and plumbing.

## Garage and vehicles

- Vehicle lifts, diagnostics, repairs, paint/upgrade workflow, registry, ownership, shared garages, and persistent vehicle storage/retrieval.

## Economy and roleplay

- Dynamic trader pricing, banking, player businesses, territory, factions, jobs, licenses, police/EMS/fire roles, and admin audit tools.

## Industry

- Farming, lumber, mining, metalworking, furniture and vehicle-part manufacturing, food/medical production, logistics, and fictional contraband production chains.
- Weapon assembly should remain gameplay-oriented and use fictional in-game parts/blueprints rather than real-world manufacturing instructions.

## Wonderland Studio

- Separate desktop developer tool for build management, XML validation, loot/trader/economy editors, later map/NPC/quest tools, and release assistance.
